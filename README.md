# Análisis de Vehículos Usados - Dashboard Interactivo

## 📋 Descripción del Proyecto
Este proyecto consiste en una aplicación web interactiva desarrollada con Streamlit que permite visualizar y analizar datos de anuncios de venta de coches usados. La aplicación ofrece gráficos dinámicos y estadísticas básicas para explorar las características de los vehículos.

## 🎯 Funcionalidades
- **Vista previa de datos**: Tabla interactiva con las primeras filas del dataset
- **Histograma del odómetro**: Visualización de la distribución del kilometraje
- **Gráfico de dispersión**: Relación entre precio y odómetro, coloreado por condición del vehículo
- **Estadísticas rápidas**: Métricas clave como total de vehículos, precio promedio, etc.
- **Controles interactivos**: Casillas de verificación para mostrar/ocultar gráficos

## 🚀 Cómo usar la aplicación
1. Abre la aplicación en tu navegador (enlace proporcionado)
2. En el panel lateral, selecciona las casillas de verificación para generar:
   - Histograma del odómetro
   - Gráfico de dispersión precio vs odómetro
3. Explora los gráficos interactivos (zoom, pan, descarga)
